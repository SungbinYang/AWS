# 회의 참가 설문조사 프로젝트

## 만든 계기
- 먼저 AWS 학습을 통한것을 적용해본 계기가 되었다. 각종 서비스들을 적용하는것을 한번 만들어보자는 생각과 현재 회사에서 회의에 대해 일일이 엑셀이나 전환통화로 조금 비효율적인 방식을 조금이나마 개선해보고자 이 어플리케이션을 만들게 되었다.

## 스펙
- 사용기술: python 3.7, HTML5, CSS3, Javascript
- Devops: Docker
- AWS: CloudFront, S3, API Gateway, Lambda, DynamoDB, SNS, EC2
